## DOS 命令

#### 1. nslookup

```txt
nslookup 域名                   查询域名对应的IP地址
```

#### 2. ping

```txt
ping www.baidu.com             测试联网情况 参数: -l 2000(每个包2000字节)

ping -t www.baidu.com          一直ping

ping -n 数字 www.baidu.com     修改ping包数量 默认为4
```

#### 3. net

```txt
net user 用户 密码 /add         添加用户(需要开权限)

net localgroup                 修改计算机上的本地组
```
